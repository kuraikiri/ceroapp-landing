# Cero — Calculadora financiera chilena

Sitio web oficial de **Cero**, la calculadora financiera más completa de Chile.

**[ceroapp.cl](https://ceroapp.cl)**

## Qué es Cero

Cero es una app Android gratuita que reúne más de 30 herramientas financieras diseñadas para Chile:

- Calculadora científica con indicadores en tiempo real (UF, UTM, USD, EUR)
- Simulador de sueldo líquido, finiquito y horas extra
- Simulador de créditos de consumo e hipotecarios
- Conversor de divisas (35+ monedas) con gráficos históricos
- Herramientas de inversión, negocios y beneficios sociales
- Widgets de escritorio con datos actualizados
